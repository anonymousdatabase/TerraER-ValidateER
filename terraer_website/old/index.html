TerraER

TerraER é um aplicativo gratuito que facilita o aprendizado de modelagem de dados. A ferramenta é inicialmente voltada ao meio acadêmico, mais especificamente no auxílio ao aprendizado de disciplinas de modelagem de banco de dados.

O software funciona como um editor e possui algumas funcionalidades básicas:

  • Construção do modelo de entidade e relacionamento (ER)
  • Interface simples
  • Facilidade de aprendizado
  • Geração de arquivos em formato XML
  • Exportação para área de transferência
  • Impressão

Mais importante, a ferramenta TerraER é bastante funcional e facilita o aprendizado de modelagem de dados, visto que os modelos são fáceis de serem criados, modificados e armazenados.

Concluido o modelo conceitual com TerraER, você está a um passo de criar seu modelo em seu SGBD de preferência, tal como MySql, Oracle, SQL Server, DB2, etc.
